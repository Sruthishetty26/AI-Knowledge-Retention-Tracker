# AI-Powered Knowledge Retention Tracker

## Overview

AI-Powered Knowledge Retention Tracker is an intelligent educational platform designed to measure and improve student learning retention.

The system uses Artificial Intelligence to generate dynamic questions, evaluate learner responses, identify weak concepts, and recommend personalized revision schedules.

## Features

- Dynamic AI-generated questions
- Multiple Choice Questions (MCQs)
- Descriptive answer evaluation
- Retention score calculation
- Weak topic identification
- Personalized revision recommendations
- Learning progress tracking
